class giblets.policy.Blacklist¶

policy that activates all Components that have not been specifically disabled.

disable_component(component)¶

Do not allow the component specified to be activated for this ComponentManager.

component may be a full class name string ‘foo.bar.Quux’ a Component type or an instance of a Component.

enable_component(component)¶

Allow the component specified to be activated for this ComponentManager.

component may be a full class name string ‘foo.bar.Quux’ a Component type or an instance of a Component.

is_component_enabled(component)¶
returns True unless the component specified has been disabled.

class giblets.policy.Patterns¶

A policy which enables and disables components based on an ordered list of wildcard patterns like foo.bar.* etc. First match is taken.

append_pattern(pattern, enable)¶
helper to add a pattern to the end of the list of patterns.
build_pattern(pattern, enable)¶

create an entry suitable for insertion into the patterns list of this manager.

If pattern is a string, it is treated as a wildcard patten like foo.bar.*

Otherwise, it is assumed pattern is a compiled regular expression.

eg: pat = mgr.build_pattern(‘foo.*’, True) mgr.patterns.insert(0, pat)
